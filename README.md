# Projeto-Site-Devops
Primeiro site que criei usando as tecnlogias: Html, Java Script e CSS. O site é para reprensentar uma empresa de tecnologia fictícia para um prejeto de DevOps. Tem a funcionalidade de mandar mensagens por e-mail, mas não apresenta responsividade para todas as resoluções
